# Notebook Series

This folder contains the consolidated five-part notebook series for the Python + ORE workflow:

1. `01_python_to_ore_swig_dataclasses.ipynb`
2. `02_ore_snapshot_capabilities.ipynb`
3. `03_curve_calibration_and_lgm_params.ipynb`
4. `04_python_lgm_xva_model.ipynb`
5. `05_joint_python_and_ore_workflow.ipynb`

The notebooks share a thin support layer in `series_helpers.py` and are generated by `build_series.py`.

To regenerate the notebooks after editing the builder:

```bash
python3 notebook_series/build_series.py
```

The helpers in this folder prefer local vendored `Examples/` and local `parity_artifacts/` from the standalone
`PythonOreRunner` repo. If a notebook needs a live ORE binary or non-vendored Engine inputs, set `ENGINE_REPO_ROOT`
to a local `Engine` checkout.
